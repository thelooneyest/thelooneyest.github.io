Maxwell Fashions website

Project created with:

  • HTML
  • CSS
  • JavaScript

This project was created for Maxwell Fashions, a clothing design startup. It was mostly created to showcase their athletic wear. But, they also needed sections to show off portfolio and give information about the company.

I created a straight forward experience with mostly black and white with the color coming from the photos that were provided by the client. The website is responsive to work on both desktop and mobile devices.
