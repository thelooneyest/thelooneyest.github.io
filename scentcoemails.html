Scentco Inc. E-mail Template

Project created with:

  • MailChimp
  • HTML
  • CSS

This project was created for Scentco Inc., a company specializing in scented and unscented toys and school supplies. I created this email template in MailChimp using custom HTML and CSS.

The purpose of these emails is to show off the products to potential retail buyers. Each email contains links to product information sheets and product videos when available. Pertinant information on each product including features and prices are listed in the email in text format so that they are easily viewable by as many different users as possible.
