Scentco Sell-A-Thon website

Project created with:

  • Shopify
  • HTML
  • CSS

This project was created for Scentco Inc., a company specializing in scented and unscented toys and school supplies. I created this site in Shopify using a custom-modified version of the free Debut theme.

This is a one page site that is used as an online store for school fundraisers. The purpose of the site is to be very easy to use and easy to shop on. Because of this, I kept the store on the homepage and didn't distract with any other information. Each product has it's own page, but it's easy to get back to the homepage so that the focus is still on the products and the goal of fundraising.
